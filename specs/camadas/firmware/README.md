# Camada Firmware (ESP32)

## 1. Descrição
Esta camada define o comportamento embarcado do leitor ESP32 com RC522 para provisionamento de TAG e validação de acesso.

## 2. Objetivo
- Garantir operação segura da TAG no dispositivo de borda.
- Assegurar interoperabilidade com relayer no fluxo de provisionamento e challenge-response.

## 3. Topicos da camada
- `topicos/01-escopo-e-responsabilidades.md`
- `topicos/02-requisitos-funcionais-da-camada.md`
- `topicos/03-requisitos-nao-funcionais-da-camada.md`
- `topicos/04-fluxos-e-criterios-de-aceite.md`

## 4. Resultado esperado de aprendizagem
Ao concluir esta trilha, o aluno deve ser capaz de reproduzir e explicar o comportamento de leitura/escrita NFC e segurança embarcada sem depender de ajustes manuais de modo.
