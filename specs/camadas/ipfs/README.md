# Camada IPFS

## 1. Descrição
Esta camada define os requisitos de armazenamento descentralizado e recuperação de documentos DID por CID.

## 2. Objetivo
- Garantir persistência de documentos off-chain.
- Assegurar recuperação por CID para os fluxos de autenticação.

## 3. Topicos da camada
- `topicos/01-escopo-e-responsabilidades.md`
- `topicos/02-requisitos-funcionais-da-camada.md`
- `topicos/03-requisitos-nao-funcionais-da-camada.md`
- `topicos/04-fluxos-e-criterios-de-aceite.md`

## 4. Resultado esperado de aprendizagem
Ao concluir esta trilha, o aluno deve operar o node IPFS local e validar persistência de CIDs no ciclo completo do projeto.
