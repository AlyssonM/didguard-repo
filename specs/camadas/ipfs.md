# Camada - IPFS

## Descrição
Especificação da camada de armazenamento descentralizado para DID Documents.

## Requisitos aplicaveis
- RF-004, RF-005, RF-010
- RNF-005

## Comportamento esperado
1. Armazenar DID Document e retornar CID.
2. Permitir recuperação de documento por CID.
3. Manter persistência via volume Docker.
