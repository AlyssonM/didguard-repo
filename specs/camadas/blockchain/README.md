# Camada Blockchain

## 1. Descrição
Esta camada especifica a lógica on-chain de identidade, credenciais e autorização de acesso.

## 2. Objetivo
- Definir políticas de acesso auditaveis e verificáveis em contrato.
- Assegurar consistência do mapeamento DID-CID e credenciais.

## 3. Topicos da camada
- `topicos/01-escopo-e-responsabilidades.md`
- `topicos/02-requisitos-funcionais-da-camada.md`
- `topicos/03-requisitos-nao-funcionais-da-camada.md`
- `topicos/04-fluxos-e-criterios-de-aceite.md`

## 4. Resultado esperado de aprendizagem
Ao concluir esta trilha, o aluno deve explicar regras de autorização on-chain e validar cenários de concessão, expiração e revogação.
