# Camada Workstation

## 1. Descrição
Esta camada define o setup da estação de trabalho usada para desenvolvimento, testes e operação do DIDGuard.

## 2. Objetivo
- Garantir que todos os alunos partam de um ambiente local consistente.
- Reduzir falhas de execução por divergência de ferramentas.

## 3. Topicos da camada
- `topicos/01-escopo-e-responsabilidades.md`
- `topicos/02-requisitos-funcionais-da-camada.md`
- `topicos/03-requisitos-nao-funcionais-da-camada.md`
- `topicos/04-fluxos-e-criterios-de-aceite.md`
- `topicos/05-guia-instalacao-e-configuracao.md`

## 4. Resultado esperado de aprendizagem
Ao concluir esta trilha, o aluno deve preparar uma workstation do zero, instalar ferramentas e extensões do projeto, e validar a execução básica de todos os módulos.
