# Camada - Workstation (Ambiente de Desenvolvimento)

## Descrição
Especificação da camada de preparação da estação de trabalho para execução do DIDGuard em laboratório.

## Requisitos aplicáveis
- RNF-005, RNF-007, RNF-008

## Comportamento esperado
1. Instalar ferramentas base do ambiente de desenvolvimento.
2. Instalar e configurar a IDE Antigravity.
3. Instalar extensões locais (`.vsix`) disponíveis no repositório.
4. Instalar dependências de software por módulo.
5. Validar o setup com comandos de smoke test.
