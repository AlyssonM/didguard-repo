# Camada RelayerNode

## 1. Descrição
Esta camada define o serviço de integração entre firmware, blockchain, IPFS e frontend.

## 2. Objetivo
- Garantir contratos de API consistentes entre todas as camadas.
- Orquestrar escrita on-chain, persistência auxiliar e validacoes de acesso.

## 3. Topicos da camada
- `topicos/01-escopo-e-responsabilidades.md`
- `topicos/02-requisitos-funcionais-da-camada.md`
- `topicos/03-requisitos-nao-funcionais-da-camada.md`
- `topicos/04-fluxos-e-criterios-de-aceite.md`

## 4. Resultado esperado de aprendizagem
Ao concluir esta trilha, o aluno deve ser capaz de operar a API do relayer, validar fluxos de segurança e recuperar estado após recreate.
