# Frontend - Escopo e Responsabilidades

## 1. Descrição
Define responsabilidades da camada de interface para operação e visualizacao do DIDGuard.

## 2. Objetivo
- Delimitar o que deve ser realizado no cliente web.
- Garantir separacao entre visualizacao e lógica de integração.

## 3. Escopo funcional da camada
1. Iniciar provisionamento e acompanhar status.
2. Capturar biometria facial no navegador.
3. Permitir administração de acesso e revogação.
4. Exibir timeline de eventos de segurança.
5. Buscar usuários e dispositivos para operação.

## 4. Fora de escopo da camada
1. Decisão final de autorização.
2. Persistência de políticas de acesso.
3. Assinatura/transação blockchain direta pelo navegador.

## 5. Bibliografia
- React docs.
- Next.js docs.
- UX para sistemas de operação crítica.
