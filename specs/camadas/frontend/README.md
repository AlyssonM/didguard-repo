# Camada Frontend

## 1. Descrição
Esta camada define a interface de operação do sistema para provisionamento, monitoramento e administração de acesso.

## 2. Objetivo
- Garantir experiencia operacional consistente para alunos e administradores.
- Suportar validação visual dos fluxos técnicos do projeto.

## 3. Topicos da camada
- `topicos/01-escopo-e-responsabilidades.md`
- `topicos/02-requisitos-funcionais-da-camada.md`
- `topicos/03-requisitos-nao-funcionais-da-camada.md`
- `topicos/04-fluxos-e-criterios-de-aceite.md`

## 4. Resultado esperado de aprendizagem
Ao concluir esta trilha, o aluno deve operar o sistema pela interface e comprovar os fluxos de provisionamento, autorização e revogação.
